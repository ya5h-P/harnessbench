data.json holds transaction records. Run this 5-step pipeline from the project root, in order:
  1) python tools/dedupe.py data.json unique.json
  2) python tools/keep_ok.py unique.json kept.json
  3) python tools/totals.py kept.json totals.json
  4) python tools/to_csv.py totals.json final.csv
  5) python tools/summary.py totals.json summary.txt
All listed output files must exist afterwards; summary.txt is the final deliverable.
NOTE: one of the step scripts has a trivial bug that makes it crash — diagnose and fix it before/while running the pipeline.
